# LinkedIn Project Entry

## Title

PowerPoint Accessibility Checker

## Description

Built a Python FastAPI backend that analyzes PowerPoint files for accessibility issues including missing titles, missing image alt text, list formatting problems, and low color contrast. The tool parses Office Open XML directly and can generate remediated presentation output with improved contrast and fallback alt text.

Python, FastAPI, Accessibility, WCAG, PowerPoint, Office Open XML, Backend Development, Document Automation
